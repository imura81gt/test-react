コンポーネントを階層で扱う
================================

演習
----------------

FilterableProductTableコンポーネントを作成して
FilterableProductTableの子コンポーネントにSearchBarコンポーネントを作成して
ReactDOM.renderでレンダリングする。


確認方法
----------------

chromeの場合は [React Developer Tools][ReactDeveloperTools] をインストールする。
Developer Toolsを起動すると `react` タブが出てくるので確認ができる。

[React Developer Tools]:https://chrome.google.com/webstore/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi
